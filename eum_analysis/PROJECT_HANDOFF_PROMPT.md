## 1. 프로젝트 한 줄 요약

**경남 13개 인구감소/관심지역의 카드매출과 생활인구를 결합해, 생활인구가 지역 소비로 전환되는 정도를 진단하고 지역별 침체 유형과 맞춤형 정책 처방을 제안하는 데이터 기반 상권 진단 모델 `이음(E-um)` 프로젝트.**

핵심 차별점은 단순히 “어디가 침체됐는가”를 보는 것이 아니라, **왜 침체됐는가**를 `MI`, `CPC`, `STI` 등으로 나누어 설명하는 것이다.

## 2. 프로젝트 배경

### 2.1 문제 인식

경남 인구감소지역의 상권 문제는 단순히 인구가 적다는 사실만으로 설명하기 어렵다. 어떤 지역은 생활인구가 있으나 소비가 지역 안에서 충분히 발생하지 않고, 어떤 지역은 관광/계절 유입은 강하지만 체류와 객단가가 약하며, 어떤 지역은 고령 소비 의존과 업종 집중이 동시에 나타난다.

따라서 본 프로젝트는 기존의 행정구역 평균 통계나 정성적 현장 진단을 넘어, 카드매출과 생활인구를 결합해 **생활인구 대비 소비전환 취약성**을 정량화한다.

### 2.2 제출/활용 맥락

- 대학 과제용 수행계획서 `이음_계획서_v4.docx` 작성 완료
- 경상남도 빅데이터 활용 공모전 제출 가능성을 염두에 둔 분석 구조
- 공모전 보고서/발표에서는 Phase 2 결과를 중심으로 사용하는 것이 적절함

### 2.3 벤치마킹 방향

기존 우수작처럼 좁은 범위에서 구체적인 처방까지 제시하는 방식이 유리하다. 이 프로젝트도 13개 지역 전체를 보여주되, 보고서/발표에서는 하동군, 산청군, 합천군, 의령군, 함양군 등 핵심 사례를 깊게 설명하는 전략이 적합하다.

## 4. 핵심 지표 정의

### 4.1 가장 중요한 지표

| 지표 | 정의 | 해석 |
|---|---|---|
| `MI` | `log(연간 카드매출 / 연평균 생활인구) - 13개 지역 평균` | 낮을수록 생활인구 대비 소비전환 취약 |
| `CPC` | `연간 카드매출 / 연평균 생활인구 / 12` | 낮을수록 생활인구 1인당 월 소비 흡수력 약함 |
| `STI` | `성수기(6~8월) 생활인구 / 비수기(1~3월) 생활인구 - 1` | 높을수록 관광/계절 유입 특성 강함 |

### 4.2 주의가 필요한 지표: `LRR`

기존 문서에서는 `LRR`을 “지역 자금 환류율”, “자금 유출”, “출혈”로 해석했으나 이는 부정확하다.

현재 코드 산식:

```text
LRR = 유입지가 경남인 매출 / 해당 지역 전체 매출
```

정확한 해석:

```text
LRR = 경남권 소비자 비중
```

낮은 `LRR`은 “자금이 밖으로 빠져나감”이 아니라, **해당 지역 매출 중 경남 외 소비자 비중이 높음**, 즉 **외부유입 소비 의존 특성**으로 해석해야 한다.

사용 금지 표현:

- 지역 자금 환류율
- 자금 역외 유출
- 출혈성 침체
- 자금 유출형

권장 표현:

- 경남권 소비자 비중
- 외부유입 소비 의존
- 경남 외 소비자 비중이 높음
- 지역 내 소비 기반 안정성이 낮을 가능성

### 4.3 기타 지표

| 지표 | 정의 | 해석 |
|---|---|---|
| `CDI` | 업종별 매출 비중의 Shannon entropy | 낮을수록 특정 업종 의존도 높음 |
| `NAR` | 18~23시 매출 / 전체 매출 | 낮을수록 야간 체류/상권 활동 약할 가능성 |
| `AGI` | 60대 이상 매출 비중 | 높을수록 고령 소비 의존 |
| `YR` | 20대 미만~30대 매출 비중 | 낮을수록 청년 소비 기반 약할 가능성 |
| `TREND` | 월별 매출 회귀 기울기 / 평균 매출 | 음수면 2024년 내 매출 감소 추세 |

## 5. 침체 유형 분류 체계

### 5.1 최신 유형명

| 유형 | 진단 기준 | 처방 방향 |
|---|---|---|
| 외부의존+소비전환 취약형 | `LRR` 낮음 + `MI` 낮음 | 외부 방문/생활인구를 지역 소비로 전환, 체류·객단가 제고 |
| 소비전환 취약형 | `MI`, `CPC` 낮음 | 생활인구의 지역 소비 전환, 업종 재구성, 체류시간 연장 |
| 외부유입 의존형 | `LRR` 낮음 | 외부 방문 소비 의존 구조 관리, 지역 내 소비 기반 보완 |
| 만성 노화형 | `AGI` 높음 + 매출 감소 | 고령친화 업종, 생활서비스, 의료·돌봄 연계 |
| 혈류 부족형 | 생활인구 자체 부족 | 접근성, 생활 인프라, 거점 유입 전략 |
| 상대적 양호형 | `MI`, `CPC`, `LRR` 상대적으로 양호 | 주변 취약지역과 연계 가능한 거점 |
| 경계형 (혼합) | 단일 유형으로 보기 어려움 | 추가 모니터링 및 세부 원인 분석 |

## 9. 알려진 한계 및 주의사항

### 9.1 데이터 한계

- 2024년 단년도 자료라 전년 대비 추세 분석은 불가능
- 생활인구는 13개 인구감소/관심지역에만 있어 Phase 2 범위가 13개로 제한됨
- 카드매출은 시군구 단위라 읍면동/집계구 단위 정밀 분석은 현재 불가능
- 통신사 원천 유동인구 데이터는 미보유

### 9.2 방법론 한계

- Phase 2 표본이 13개뿐이므로 K-means 결과는 확정 분류가 아니라 탐색적 유형화로 표현해야 함
- 위험점수 가중치는 정책적 판단이 들어간 설계값이므로 민감도 분석이 있으면 더 좋음
- SHAP/RandomForest를 사용할 경우도 예측 성능보다 원인 설명 보조로 제시해야 함

### 9.3 해석상 주의

가장 중요한 주의사항:

```text
LRR을 자금 유출로 해석하지 말 것.
```

낮은 `LRR`은 외부유입 소비 의존 또는 경남 외 소비자 비중이 높다는 의미다.

## 10. 다음 AI가 반드시 알아야 할 사실

1. 이 프로젝트의 핵심 지표는 `MI`다.
2. `LRR`은 기존 이름은 유지하더라도 해석은 `경남권 소비자 비중`이다.
3. 보고서 중심은 Phase 2, 즉 13개 지역 분석이다.
4. 최신 우선순위는 하동군, 산청군, 합천군, 의령군, 함양군이다.
5. 기존 문서의 `출혈`, `자금 유출`, `지역 자금 환류율` 표현은 수정 대상이다.
6. `run_analysis.py`, `run_phase2.py`는 현재 Windows 환경에서 재실행 확인 완료.
7. 다음 주요 개발 과제는 `E-um Detect`다.
8. 공모전 보고서에서는 K-means를 “탐색적 유형화”라고 표현해야 한다.
9. `PROJECT_FLOW_AND_FIXES.md`에 파일 흐름도와 수정 체크리스트가 있다.
10. 이 문서는 인수인계 본문이고, 아래 부록은 필요할 때 복사해 쓰는 프롬프트다.

```text
E-um Phase 2 결과를 바탕으로 지역별 맞춤형 상권 처방전을 작성해줘.

지역별 해석 방향:
- 하동군: MI와 CPC가 낮아 생활인구 대비 소비전환 취약. 외부유입 의존도 함께 점검.

1. 진단 한 문장
2. 핵심 근거 지표 2~3개
3. 정책 처방 3개
4. 기대효과
```
