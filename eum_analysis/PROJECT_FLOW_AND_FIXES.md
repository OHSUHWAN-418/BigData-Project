## 5. 현재 핵심 지표 해석

### 5.1 비교적 안정적인 지표

| 지표 | 의미 | 해석 |
|---|---|---|
| `MI` | 생활인구 대비 매출 전환 정도 | 낮을수록 생활인구가 소비로 전환되지 않음 |
| `CPC` | 생활인구 1인당 월평균 카드매출 | 낮을수록 소비 흡수력이 약함 |
| `STI` | 성수기 생활인구 증가율 | 높을수록 관광/계절 유입 특성이 강함 |
| `CDI` | 업종 다양성 | 낮을수록 특정 업종 의존도가 큼 |
| `NAR` | 야간 매출 비중 | 낮을수록 야간 체류/상권 활동이 약할 가능성 |
| `AGI` | 60대 이상 소비 비중 | 높을수록 고령 소비 의존도가 큼 |
| `TREND` | 월별 매출 추세 | 음수면 2024년 내 감소 추세 |

### 5.2 수정이 필요한 지표: `LRR`

현재 계산:

```text
LRR = 유입지가 경남인 매출 / 전체 매출
```

현재 문서 해석:

```text
낮은 LRR = 지역 자금 유출 / 출혈
```

수정해야 할 해석:

```text
낮은 LRR = 해당 지역 매출 중 경남 외 지역 소비자 비중이 높음
```

따라서 `LRR`은 "자금 환류율"보다 다음 표현이 더 안전하다.

- 경남권 소비자 비중
- 지역권 소비 의존도
- Local-Origin Ratio
- 내부권 소비 비중

낮은 값은 "출혈"이 아니라 다음처럼 해석해야 한다.

- 외부 유입 소비 의존도가 높음
- 광역 방문 소비 비중이 큼
- 지역 내 수요 기반이 약할 가능성

## 6. 지금 바로 수정해야 할 사항

### 6.1 최우선 수정

| 우선순위 | 수정 사항 | 이유 |
|---|---|---|
| 1 | `LRR` 명칭과 해석 수정 | 현재 보고서 논리에서 가장 큰 위험 |
| 2 | `출혈형`, `자금 유출형` 유형명 수정 | 데이터로 직접 증명되지 않는 표현 |
| 3 | 위험점수에서 `LRR` 역할 재조정 | 낮은 LRR을 위험으로만 보면 외지 소비 유입 지역을 오판 가능 |
| 4 | 문서 한글 깨짐 복구 | 제출/협업 문서로 사용 불가 |
| 5 | 코드 입출력 경로 상대경로화 | 현재 환경에서 재실행성 낮음 |

### 6.2 유형명 수정안

| 기존 유형명 | 수정 권장 유형명 | 이유 |
|---|---|---|
| 출혈+소화 복합형 | 외부의존+소비전환 취약형 | 낮은 LRR을 자금 유출로 단정하지 않기 위해 |
| 출혈성 자금 유출형 | 외부유입 의존형 | 외지 소비 비중이 높은 구조로 재해석 |
| 소화 불량형 | 소비전환 취약형 | MI/CPC 기반 해석과 잘 맞음 |
| 성장 양호형 | 상권 양호형 | 과도한 성장 표현 완화 |
| 경계형 | 경계형 | 유지 가능 |

### 6.3 보고서 문장 수정안

기존 표현:

```text
LRR이 낮아 자금이 외부로 유출되고 있다.
```

수정 표현:

```text
LRR이 낮다는 것은 해당 지역 매출에서 경남 외 소비자 비중이 높다는 뜻이다. 이는 외부 방문 소비 의존도가 높거나 지역 내 수요 기반이 상대적으로 약할 가능성을 시사한다.
```

기존 표현:

```text
함양군은 자금 유출이 가장 심각하다.
```

수정 표현:

```text
함양군은 경남권 소비자 비중이 낮고 외부 소비 의존도가 높으며, 동시에 매출 감소 추세와 계절성이 나타나 소비 기반 안정성이 낮은 지역으로 해석된다.
```

기존 표현:

```text
출혈+소화 복합형은 즉각적인 처방이 필요하다.
```

수정 표현:

```text
외부의존+소비전환 취약형은 생활인구와 외부 유입을 지역 내 소비로 전환하는 전략이 우선 필요하다.
```

## 7. 지역별 재해석 방향

| 지역 | 현재 주요 신호 | 수정된 해석 |
|---|---|---|
| 하동군 | `MI` 낮음, `CPC` 낮음, `LRR` 낮음 | 생활인구 대비 소비전환 취약 + 외부 유입 의존 |
| 함양군 | `LRR` 최저, `TREND` 최저, `STI` 높음 | 외부소비 의존 + 매출 감소 + 계절성 |
| 합천군 | `MI` 최저, `CPC` 최저 | 생활인구 대비 소비 흡수력 최약 |
| 의령군 | `MI` 낮음, `CPC` 낮음 | 지역 내 소비전환 기반 약함 |
| 산청군 | `STI` 최고, `MI` 낮음 | 관광/성수기 생활인구 증가가 소비로 충분히 전환되지 않음 |
| 남해군 | `LRR` 낮음, 관광성 가능 | 외부 관광소비 의존이 있으나 소비전환 수준 추가 점검 필요 |

## 8. 위험점수 수정 방향

현재 위험점수는 대략 다음 구조다.

```text
위험점수 =
  (-MI) * 0.30
  + (1 - LRR) * 0.25
  + AGI * 0.20
  + (-TREND * 100) * 0.15
  + (1 - CDI / CDI.max()) * 0.10
```

문제:

- `LRR`이 낮은 것을 위험으로 직접 가산하고 있음
- 외부 방문 소비가 많은 관광/경계 지역이 과도하게 위험으로 분류될 수 있음
- `CPC`가 위험점수에 직접 반영되지 않음
- `STI`가 관광 전환 문제에 직접 반영되지 않음

수정 방향:

```text
기본 위험점수 =
  MI 취약도
  + CPC 취약도
  + TREND 감소도
  + CDI 취약도
  + AGI 고령의존도
```

`LRR`은 위험점수보다는 별도 특성 지표로 분리하는 것이 안전하다.

```text
LRR 낮음 -> 외부유입 의존 특성
STI 높음 + MI 낮음 -> 관광 유입 소비전환 취약
```

## 11. 최종 판단

현재 프로젝트의 가장 강한 지점은 `MI`다.

```text
MI = log(연간 카드매출 / 평균 생활인구) - 13개 지역 평균
```

이 지표는 "사람이 있는가"가 아니라 "있는 사람이 소비로 전환되는가"를 보여준다. 따라서 공모전 보고서의 핵심 메시지는 `LRR`보다 `MI`, `CPC`, `STI`를 중심으로 잡는 것이 좋다.

권장 핵심 문장:

```text
E-um은 인구감소지역을 단순히 인구 부족 지역으로 보지 않고,
생활인구가 실제 지역 소비로 전환되는 정도를 측정해
지역별 상권 침체 원인과 처방 방향을 유형화하는 진단 모델이다.
```
