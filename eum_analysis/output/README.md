# 「이음(E-um)」 Phase 1 분석 결과 요약

> 보유 데이터(2024년 경상남도 카드 매출 3종)로 수행한 1차 분석.
> 유동인구 데이터 확보 시 Phase 2(MI 지수 추가, 4유형 분류)로 확장 예정.

---

## 1. 분석 개요

- **대상**: 경남 22개 시군구 전수
- **데이터**: 2024년 카드 매출 데이터 3종 (월별 28,414행 / 시간대별 5,205행 / 성연령별 33,622행)
- **분석 도구**: Python (pandas, scikit-learn), K-means 군집, PCA
- **산출 지표**: 6종 (LRR, CDI, NAR, AGI, YR, TREND)

## 2. 핵심 지표 정의

| 지표 | 영문 | 의미 | 해석 |
|------|------|------|------|
| **LRR** | Local Return Ratio | 지역 자금 환류율 | 낮음 = 자금 역외 유출 (출혈성) |
| **CDI** | Consumption Diversity Index | 업종 다양성 (Shannon) | 낮음 = 특정 업종 의존 (단조형) |
| **NAR** | Night Activity Ratio | 야간(18~23시) 매출 비중 | 낮음 = 통과형 상권 |
| **AGI** | Aging Index | 60대 이상 매출 비중 | 높음 = 노화형 |
| **YR** | Youth Ratio | 20~30대 매출 비중 | 낮음 = 청년 이탈 |
| **TREND** | Sales Trend | 월별 매출 추세 (정규화) | 음수 = 감소 추세 |

## 3. 군집 분석 결과 (K=3)

### 유형 ① 출혈성+노화 복합형 (7개) — 🔴 최우선 대응
**거창군, 고성군, 밀양시, 양산시, 창녕군, 하동군, 함양군**
- LRR 평균: 0.796 (낮음)
- AGI 평균: 0.227 (높음)
- 자금이 빠져나가면서 동시에 고령층 의존 심화

### 유형 ② 건강 상권 (10개) — 🟢 양호
**거제시, 김해시, 사천시, 진주시, 창원시 5개구, 통영시**
- LRR 평균: 0.945 (높음)
- AGI 평균: 0.167 (낮음)
- 자금 환류 양호 + 다양한 연령층 소비

### 유형 ③ 출혈성(자금 유출형) (5개) — 🟡 주의
**남해군, 산청군, 의령군, 함안군, 합천군**
- LRR 평균: 0.820
- CDI 평균: 1.387 (낮음, 업종 집중)
- 외부 유출 + 단조로운 업종 구조

## 4. 우선 처방 대상 TOP 5

## 5. 주요 발견

1. **함양군이 모든 위험 지표에서 최악** — LRR 65.5% (1위 최저), TREND -1.43% (감소 최악). 출혈성+노화 복합형의 전형.

2. **창원·진주 중심 상권은 견고** — 창원시 5개 구 모두 LRR 88% 이상, AGI 20% 이하로 건강 상권 분류.

3. **소비 다양성과 자금 환류는 별개 차원** — 함안군은 CDI는 낮지만(1.376) LRR은 중간 수준(0.787). 단순 단일 지표로는 침체 진단 어려움을 보여줌.

4. **남해군 특이 케이스** — LRR은 매우 낮으나(0.743) AGI는 낮음(0.181). 관광지 특성으로 외지인 소비 의존도가 높지만 청·장년층 활동은 활발.

5. **공간 패턴** — 서부 산간 군 지역(함양·하동·산청·합천)에 출혈성+노화 집중. 향후 Moran's I로 공간 자기상관 검증 가능.
