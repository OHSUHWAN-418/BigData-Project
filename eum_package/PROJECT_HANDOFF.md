## 4. 핵심 지표 정의 (총 9종)

### 4.1 Phase 1 지표 (카드 데이터만 필요)

| 지표 | 영문 풀네임 | 정의 | 해석 |
|------|------------|------|------|
| **LRR** | Local Return Ratio | 경남 내 유입 매출 / 전체 매출 | 낮음 = 자금 역외 유출 (출혈성) |
| **CDI** | Consumption Diversity Index | 13업종 매출 비중의 Shannon Entropy | 낮음 = 업종 집중 (단조형) |
| **NAR** | Night Activity Ratio | 18~23시 매출 / 전체 매출 | 낮음 = 통과형 상권 |
| **AGI** | Aging Index | 60대+ 매출 비중 | 높음 = 고령 의존 (노화형) |
| **YR** | Youth Ratio | 20~30대 매출 비중 | 낮음 = 청년 이탈 |
| **TREND** | Sales Trend | 월별 매출 회귀 기울기 (정규화) | 음수 = 감소세 |

### 4.2 Phase 2 신규 지표 (생활인구 결합 시 가능, ★핵심)

| 지표 | 영문 풀네임 | 정의 | 해석 |
|------|------------|------|------|
| **MI** ★ | Mismatch Index | log(연간 매출 / 연평균 생활인구) − 13개 평균 | **음수 = 소화 불량형 침체** |
| **CPC** | Consumption Per Capita | 연간 매출 / 연평균 생활인구 / 12 | 1인당 월 소비액 (구매력) |
| **STI** | Seasonal Tourism Index | (성수기 6~8월 생활인구) / (비수기 1~3월) − 1 | 높음 = 관광 의존 |

## 5. 침체 4유형 분류 체계

### 5.1 4유형 정의 (이음(E-um)의 핵심 차별화 포인트)

| 유형 | 진단 기준 | 처방 방향 |
|------|---------|----------|
| **혈류 부족형** (Low-Flow) | 생활인구 자체가 부족 | 접근성·교통 인프라, 랜드마크 유치 |
| **소화 불량형** (Low-Conversion) | MI 음수 (유동은 있으나 소비 부진) | 업종 재구성, 야간 체류 유도 |
| **만성 노화형** (Aging) | AGI 높음 + 매출 감소 | 생활밀착·의료 연계 업종 강화 |
| **출혈성** (Leakage) | LRR 낮음 (자금 역외 유출) | 지역화폐 확대, 온라인 연계 |

## 7. 작업 이력 (시간순)

1. **초기 계획서 검토** — 원본 계획서를 받아 4가지 보강 항목(필요성/방안/관련 데이터/연관관계) 정리
2. **공모전 적합도 분석** — 평가 기준 매칭, 약점(독창성) 식별
3. **대상작 벤치마킹** — 「백데백승」 분석, 4가지 수정 방향 도출 (범위 좁힘, 알고리즘 고도화, 산출물 구체화, 분석 사슬 명확화)
4. **차별화 메시지 구체화** — "경남 상권의 청진기" 슬로건, 4유형 분류 체계, MI/LRR 지표 명명
5. **계획서 v2 작성** — 보강된 내용 모두 포함, 8페이지+ (학교 서식 위반)
6. **카드 데이터 3종 수령** — 즉시 EDA 가능 여부 분석
7. **유동인구 데이터 수급 시도** — 통신사 데이터는 방문 신청 필요로 포기, 대안 검토
8. **여러 인구/이동 데이터 검토** — SGIS, 이동 통계 등은 부적합 판단
9. **KOSIS 생활인구 데이터 발견** — 이게 진짜 유동인구! Phase 2 가능해짐
10. **Phase 1 분석 실행** — 22개 시군구 × 6지표 (유동 데이터 없이)
11. **Phase 2 분석 실행** — 13개 인구감소지역 × 9지표 (MI 포함 4유형 분류)
12. **계획서 v3 작성** — 3페이지 압축 버전 (학교 서식 준수)
13. **계획서 v4 작성** — **원본 양식 충실 재현 + 분량 제약 없음** (현재 최종본)
